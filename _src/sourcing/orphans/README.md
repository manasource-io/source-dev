# Orphan sourcing records

This directory holds migrated sourcing artifacts that do not currently map to a publishable resource in `resources/**`.

An orphan record should remain here only when one of the following is true:

- the old target no longer exists in the public corpus
- the target has not been promoted into a publishable resource yet
- the correct canonical destination still requires manual review

Current known orphan example:

- `info-fiber/`
