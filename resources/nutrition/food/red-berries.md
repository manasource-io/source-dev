This entry has been split into individual resources for each berry type. Please use the following instead:

- **Blueberries** (`food/blueberries`): highest anthocyanin density, cognitive protection, cancer prevention via pterostilbene
- **Strawberries** (`food/strawberries`): highest vitamin C of any berry, cardiovascular benefits, iron absorption
- **Raspberries** (`food/raspberries`): highest fiber, ellagic acid and urolithins for mitophagy and cancer prevention
- **Cranberries** (`food/cranberries`): A-type proanthocyanidins for UTI prevention
